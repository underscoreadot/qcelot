# Qcelot
A Discord queue tracker and notifier for games on the Hypixel network!

## Features
* Check the current player count for most games
* Track the player count for a game and receive notifications when it reaches a threshold
* Set default games on a per server and per mode basis

## Installation
Qcelot requires Node.js v18 or later. Consult the [Discord Developer Documentation](https://docs.discord.com/developers/quick-start/getting-started) for information on creating and setting up an application. A Hypixel API key (`HYPIXEL_API_KEY`) is also required and must be supplied in your `.env` file. Game icons are not included in the repository and must be supplied manually.
